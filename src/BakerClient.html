Baker Client is a small utility used in client target machines to register them to the marketplace. Baker Client can manage the requested applications by the marketplace to the target machine.

Starting Baker Client

Use the following command to start the client manually:

java -jar '-Dmarketplace_api_endpoint=http://83.212.106.218:8080/baker/services/api/repo' '-Dbaker_client_owner_id=34' '-Dbaker_client_owner_username=dev' '-Dbaker_client_host_fqdn=example.com' target/gr.upatras.ece.nam.baker.client-0.0.1-SNAPSHOT.war

Replace baker_client_owner_id with your Marketplace user ID

Replace baker_client_owner_username with your Marketplace username

Replace baker_client_host_fqdn with your server FQDN

After a few minutes of operation, Baker Client will successfully register itself to the marketplace, under your account. You can see it if is registered successfully by visiting your Registered Resources

Using Baker Client

By default baker client listens on port 13001 for local requests. To manage it, open a browser and go to http://localhost:13001
